# Project Plan

### Project Description

ArtCart will be an art supply inventory web app that can help users categorize and keep track of various art supplies that they have or would like to have in the future. Two distinct functions include the inventory (for art supplies users already have) and the shopping list (for art supplies they want to have in the future).

### Planned Features

1. Art supply inventory: Allows users to create, edit, and delete art supply items.
2. Supply filtering: Filters given art items by category (medium type, brand, store origin, etc.).
3. Shopping list: Lets users create, edit, and delete items they want to buy in the future.
4. Responsive layout: Can be viewed from desktops, tablets, and mobile devices.
5. Product sorting: Displays art supply inventory according to supply categories.

### Tech Stack

I am hoping to use the MERN stack for this web application. This includes:
- MongoDB - Database storing
- Express.js - Backend framework that works with Node.js
- React - Frontend library
- Node.js - JavaScript runtime environment
